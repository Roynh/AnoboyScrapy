# AnoboyScrapy

Script Python untuk mendapatkan link embed dari scrape website anoboy (https://62.182.83.93/).

## Description

Dibuat dengan Scrapy.

## Getting Started

### Dependencies

* python3.6+.
* Scrapy

### Installing

* install Scrapy
* pip install Scrapy

### Executing program

* python3 search_anoboy.py
